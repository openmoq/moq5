# Archived API Sketches

These examples are **pre-implementation design sketches** from the API
exploration phase. They reference APIs that do not yet exist (subscribe,
publish, announce, SimPair, relay forwarding, etc.).

They informed 13 rounds of external API review and the decisions captured
in `docs/API_DECISIONS.md`. They are preserved here as design artifacts,
not as current usage examples.

**For current working examples, see `examples/session/`.**
